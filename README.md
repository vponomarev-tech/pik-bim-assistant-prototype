# PIK BIM Assistant (прототип)

AI-ассистент по BIM-технологиям на основе RAG и статей ПИК с Хабра.

## Что это?

Чат-бот, который отвечает на вопросы о BIM-технологиях, опираясь на реальные статьи компании ПИК. Бот не придумывает — ищет ответ в документах и показывает источники.

## Стек

- **LLM**: GigaChat (Сбер)
- **Embeddings**: intfloat/multilingual-e5-large
- **Vector Store**: FAISS
- **Framework**: LangChain
- **UI**: Streamlit
- **Язык**: Python 3.10

## Как работает

1. Тексты статей из `./data/` разбиваются на чанки
2. Каждый чанк превращается в вектор (E5 embeddings)
3. Вопрос пользователя векторизуется
4. FAISS ищет 5 ближайших чанков
5. Чанки + вопрос отправляются в GigaChat
6. GigaChat формирует ответ только на основе контекста
